¿Qué es pasaportecovid.app?

Esta sencilla aplicación web sirve para tener a mano el PDF que utilizamos como pasaporte COVID.

Una vez que subes el PDF, crea un acceso directo para que se genere un icono en la pantalla de inicio de tu móvil. De esa manera tendrás acceso rápido al pasaporte, sin tener que pensar dónde lo tienes guardado.
