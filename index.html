Seguridad

Aunque utilizamos el texto "Subir Pasaporte" ningún archivo se sube a ningún servidor. Cualquier PDF o imagen seleccionada se almacena en tu propio dispositivo, en una base de datos local llamada localStorage.

De esta manera, el archivo nunca se sube a Internet, sólo se almacena en tu dispositivo.

¿Qué es pasaportecovid.app?

Esta sencilla aplicación web sirve para tener a mano el PDF que utilizamos como pasaporte COVID.

Si ya lo tienes en tu wallet o has creado un acceso directo al pasaporte, ya está. No necesitas nuestra ayuda.

Si tienes que dar varias vueltas para encontrar tu pasaporte, te lo ponemos más fácil.

Sube tu pasaporte en formato PDF o imagen y crea un acceso directo. Pum, tu pasaporte como si fuera una app en tu móvil. Siempre a mano.
